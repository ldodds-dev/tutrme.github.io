
# TutrMe AI Learning Intelligence Platform

Live showcase: https://ldodds-dev.github.io/tutrme.github.io/#learning

<img width="1536" height="1024" alt="tutrme-cover" src="https://github.com/user-attachments/assets/ec2b4c9b-d985-4007-bcb5-81616ddf234c" />

This AI project is intended to showcase TutrMe, an education-friendly learning intelligence platform. The experience is designed around calm visuals, supportive learning signals, and a sensory-friendly interactive demo.

This project is a standalone showcase with its own audience, use case, and product story.

## Project Purpose

TutrMe presents an AI-powered learning support concept for learners, tutors, and instructors. The showcase focuses on helping users understand learning progress, organize thinking, reflect on performance, and follow adaptive study paths.

## Key Features

- Branded TutrMe cover page with learner-centered visual design
- TutrMe logo and brand system section
- Sensory-friendly live 3D learning map
- Comfort controls for gentle motion and visual softness
- Learning confidence indicator with adaptive support messaging
- Responsive layout for desktop and mobile browsers
- Static site architecture with no backend requirement

## Live Demo Sections

- Cover Page: Branded TutrMe hero visual
- Brand System: Logo-based visual identity and design direction
- 3D Demo: Calm learning map with motion controls
- Learning Model: Adaptive paths, support signals, and sensory-aware UI framing

## Technology Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas-based 3D-style learning map
- Static assets stored locally in `assets/`

No framework, build step, server runtime, or database is required.

## Run Locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 8001
```

Then open:

```text
http://localhost:8001/
```

## Deploy With GitHub Pages

1. Add the TutrMe files to the GitHub repository.
2. Make sure `index.html` is at the repository root.
3. Commit and push the files.
4. In GitHub, open repository settings.
5. Go to Pages.
6. Choose the branch and root folder to publish.
7. Save the Pages settings.

The published project URL should look like:

```text
https://ldodds-dev.github.io/tutrme.github.io/
```

The learning section can be linked directly with:

```text
https://ldodds-dev.github.io/tutrme.github.io/#learning
```

## Accessibility And Comfort

TutrMe is designed to feel calm and education-friendly:

- Gentle color palette
- Low-noise layout
- Readable spacing
- Motion toggle for the demo
- Visual softness control
- Screen-reader-friendly page title
- Responsive mobile layout

## Privacy Notes

This is a static showcase. The demo runs in the browser and does not submit learner data, store user input, or require a backend service.
